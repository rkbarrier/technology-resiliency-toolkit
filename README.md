# Technology Resiliency Toolkit

## Overview

This repository contains original examples demonstrating technology resiliency, business continuity planning, disaster recovery, and operational readiness practices commonly used within enterprise organizations.

The examples reflect industry best practices and are intended to demonstrate planning, documentation, and governance techniques without including confidential or proprietary information.

* Disaster Recovery Plan
* Recovery Strategy

* Recovery Time Objective (RTO) Examples
* Recovery Point Objective (RPO) Examples
* Tabletop Exercise Guide
* Operational Readiness Checklist
